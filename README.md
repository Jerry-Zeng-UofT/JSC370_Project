## Toronto Crime Analysis Project

- The Final project website and the report are in the docs/. 

### Description of the Project
- This project studies crime patterns across Toronto neighbourhoods from 2014 to 2025 using data from the City of Toronto Open Data Portal. The analysis looks at how crime rates differ across neighbourhoods, how they changed over time, and whether neighbourhoods can be grouped into meaningful crime profiles. It also tests how well past crime patterns can predict future neighbourhood crime rates by comparing several statistical and machine learning models. The website shows interactive visualizations that highlight the main findings, while the full analysis is presented in the downloadable report.

### Instruction of the project repo
- The website only shows a home page and interactive visualizations in separate pages. The Analysis and the whole project are in the report.

- The raw code are given in qmd in the root directory. _quarto.yml tells which files are the source code of the final website. EDA_Analysis_notebook.qmd is the raw code for EDA and analysis process, and EDA_Analysis_notebook.html is only used to create the tables and static visualizations needed for the report.

- The original dataset is obtain via API in the EDA_Analysis_notebook.qmd. The data folder contains cleaned dataset and summerized dataset for interactive visualizations.
